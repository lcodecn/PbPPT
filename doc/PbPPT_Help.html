概述

PbPPT 是一个纯 PureBasic 实现的 PowerPoint 文件操作库，无需安装 Microsoft Office 或任何第三方依赖，即可创建和读取 PowerPoint pptx 文件。

该库参考 Python 的 python-pptx 项目（v1.0.2）移植而来，使用 PureBasic 内置的 XML 和 Packer 库实现。

注意：使用 PbPPT 库前，需要在代码开头添加 XIncludeFile "PbPPT.pb" 以引入库文件，并调用 PbPPT_Init() 进行初始化。

主要功能

  • 创建和读取 PowerPoint pptx 文件
  • 添加文本框、自动形状、图片、表格、图表、连接符
  • 设置字体样式（名称、大小、颜色、粗体、斜体、下划线）
  • 设置形状样式（纯色填充、无填充、线条格式、旋转）
  • 多段落文本和富文本Run支持
  • 表格单元格读写、行高列宽设置
  • 图表数据系列和分类标签设置
  • 文档属性设置（标题、作者、主题等）
  • 幻灯片尺寸设置（标准4:3、宽屏16:9、自定义）
  • 图片去重优化，减少文件大小

演示文稿操作

PbPPT_Init()
初始化 PbPPT 库。在使用任何 PbPPT 函数之前必须调用此函数。
; 初始化PbPPT库 PbPPT_Init()
PbPPT_Create()
创建一个新的 PowerPoint 演示文稿。创建后可添加幻灯片和形状。
返回值： 成功返回 #True，失败返回 #False。
PbPPT_Init() If PbPPT_Create() ; 演示文稿创建成功 PbPPT_AddSlide(0) PbPPT_Save("output.pptx") PbPPT_Close() EndIf
PbPPT_Open(filename.s)
打开一个现有的 PowerPoint 文件。可以读取文件信息并修改后另存。
参数类型说明
filenameStringpptx文件的完整路径
返回值： 成功返回 #True，失败返回 #False。
PbPPT_Save(filename.s)
将演示文稿保存为 pptx 文件。该函数会生成符合 Office Open XML 标准的 pptx 文件。
参数类型说明
filenameString输出文件的完整路径（.pptx 后缀）
返回值： 成功返回 #True，失败返回 #False。
If PbPPT_Save("output.pptx") MessageRequester("成功", "文件已保存！") Else MessageRequester("错误", "保存失败！") EndIf
PbPPT_Close()
关闭当前演示文稿，释放所有相关资源。在保存后应调用此函数。
PbPPT_GetSlideCount()
获取当前演示文稿中的幻灯片数量。
返回值： 幻灯片数量（整数）。

返回值： 幻灯片索引（整数），从0开始。
; 添加一张空白幻灯片 slideIdx = PbPPT_AddSlide(0)
PbPPT_SetSlideWidth(width.i)
设置幻灯片宽度。单位为EMU（English Metric Units）。
参数类型说明
widthInteger幻灯片宽度（EMU）
PbPPT_SetSlideHeight(height.i)
设置幻灯片高度。单位为EMU。
参数类型说明
heightInteger幻灯片高度（EMU）
PbPPT_GetSlideWidth()
获取当前幻灯片宽度（EMU）。
返回值： 幻灯片宽度（整数，EMU）。
PbPPT_GetSlideHeight()
获取当前幻灯片高度（EMU）。
返回值： 幻灯片高度（整数，EMU）。

文本操作

PbPPT_SetShapeText(shapeId.i, text.s)
设置形状的文本内容。使用换行符（#LF$）分隔多个段落。
参数类型说明
shapeIdInteger形状ID
textString文本内容，#LF$分隔段落
; 设置多段落文本 PbPPT_SetShapeText(tbId, "第一段" + #LF$ + "第二段" + #LF$ + "第三段")
PbPPT_SetFont(shapeId.i, paraIdx.i, runIdx.i, name.s, size.i, bold.b, italic.b, underline.b, color.s)
设置形状中文本的字体样式。
参数类型说明
shapeIdInteger形状ID
paraIdxInteger段落索引（从1开始）
runIdxIntegerRun索引（从1开始）
nameString字体名称，如 "微软雅黑"
sizeInteger字号（磅），如 28
boldBoolean是否粗体
italicBoolean是否斜体
underlineBoolean是否下划线
colorString字体颜色（6位十六进制），如 "FF0000"
; 设置第1段第1个Run为微软雅黑28号粗体蓝色 PbPPT_SetFont(tbId, 1, 1, "微软雅黑", 28, #True, #False, #False, "2E75B6")

参数类型说明
shapeIdInteger形状ID
paraIdxInteger段落索引（从1开始）
textStringRun文本内容
nameString字体名称
sizeInteger字号（磅）
boldBoolean是否粗体
italicBoolean是否斜体
underlineBoolean是否下划线
colorString字体颜色（6位十六进制）

单位转换

PbPPT 使用 EMU（English Metric Units）作为内部单位。1英寸 = 914400 EMU，1厘米 = 360000 EMU，1磅 = 12700 EMU。

PbPPT_InchesToEmu(inches.f)
将英寸转换为EMU。
emu = PbPPT_InchesToEmu(1.0) ; 返回 914400 emu = PbPPT_InchesToEmu(2.5) ; 返回 2286000
PbPPT_CmToEmu(cm.f)
将厘米转换为EMU。
emu = PbPPT_CmToEmu(2.54) ; 返回 914400 (1英寸)
PbPPT_PtToEmu(pt.f)
将磅转换为EMU。
emu = PbPPT_PtToEmu(1.0) ; 返回 12700 emu = PbPPT_PtToEmu(2.0) ; 返回 25400
PbPPT_EmuToInches(emu.i)
将EMU转换为英寸。
返回值： 英寸值（浮点数）。

示例代码

示例1：创建带文本和形状的演示文稿

XIncludeFile "PbPPT.pb" PbPPT_Init() If PbPPT_Create() slideIdx = PbPPT_AddSlide(0) ; 添加标题 titleId = PbPPT_AddTextbox(slideIdx, PbPPT_InchesToEmu(1), PbPPT_InchesToEmu(0.5), PbPPT_InchesToEmu(8), PbPPT_InchesToEmu(1.5)) PbPPT_SetShapeText(titleId, "PbPPT 示例") PbPPT_SetFont(titleId, 1, 1, "微软雅黑", 28, #True, #False, #False, "2E75B6") ; 添加矩形 rectId = PbPPT_AddShape(slideIdx, #PbPPT_PrstRectangle$, PbPPT_InchesToEmu(2), PbPPT_InchesToEmu(3), PbPPT_InchesToEmu(6), PbPPT_InchesToEmu(2)) PbPPT_SetFillSolid(rectId, "4472C4") PbPPT_SetShapeText(rectId, "蓝色矩形") PbPPT_SetFont(rectId, 1, 1, "微软雅黑", 18, #True, #False, #False, "FFFFFF") PbPPT_Save("output.pptx") PbPPT_Close() EndIf

示例2：创建带图表的演示文稿

XIncludeFile "PbPPT.pb" PbPPT_Init() If PbPPT_Create() slideIdx = PbPPT_AddSlide(0) ; 添加柱状图 chartId = PbPPT_AddChart(slideIdx, #PbPPT_ChartTypeColumnClustered, PbPPT_InchesToEmu(0.5), PbPPT_InchesToEmu(1.5), PbPPT_InchesToEmu(9), PbPPT_InchesToEmu(5.5)) PbPPT_SetChartTitle(chartId, "季度销售数据") ; 添加分类 PbPPT_AddChartCategory(chartId, "Q1") PbPPT_AddChartCategory(chartId, "Q2") PbPPT_AddChartCategory(chartId, "Q3") PbPPT_AddChartCategory(chartId, "Q4") ; 添加数据系列 PbPPT_AddChartSeries(chartId, "产品A", "120,200,150,300") PbPPT_AddChartSeries(chartId, "产品B", "80,150,220,250") PbPPT_SetChartHasLegend(chartId, #True) PbPPT_Save("chart_output.pptx") PbPPT_Close() EndIf
